Description

The haskell-rl project is a repository on the Target Git Enterprise server within the RedOptHaskell organization. It was first created as a central storage point for the code created as a group of us did various exercises from the book: Reinforcement Learning: an Introduction, by Sutton & Barto, as part of a reading group led by Ramki, which currently meets every Thursday from 4-5pm in room #201 at the Target Sunnyvale location at 100 Mathilda Place, Sunnyvale, CA, USA.

Examples

The following examples are available:

  • tictactoe
  • rentalcars
  • gambler
  • blackjack

Each example above was created from a literate Haskell source file, which you can find in the app directory of this repository, and contains:

  • Haskell source code,
  • links to relevant external resources, and
  • example output

for a particular problem.
